.. _glossary:

术语表
======

本章节包含 GRPSv1 通用规则包标准 (GRPS) 相关的术语和定义。


.. glossary::

    类
        用水系搭建出的模型，用于代替原本的 :term:`骰系` 说法，以避免歧义。更多内容待补充。

    规则包
        一组包含规则和元数据的文件集合，遵循 :term:`GRPS` 标准，用于定义和执行特定的业务逻辑。

    规则
        规则包中的基本单元，定义了特定的条件和操作，用于实现业务逻辑。

    规则书
        特指纸质发行或者 pdf 格式发行的规则书文件，用于传统的桌面角色扮演游戏 (:term:`TRPG`) 中的规则说明。

    GRPS
        通用规则包标准 (Generic Rule Package Standard) 的缩写，是一种用于定义和交换规则包的开放标准。

    GRPSv1
        通用规则包标准的第一个版本，定义了规则包的基本结构和规范。

    骰系
        特指并未遵守水系 :term:`类` 规范的规则系统，用于传统的桌面角色扮演游戏 (:term:`TRPG`) 中的规则说明。

    TRPG
        桌面角色扮演游戏 (Tabletop Role-Playing Game) 的缩写，是一种玩家通过口头叙述和角色扮演进行的游戏形式，通常使用规则书和骰子来决定游戏中的事件和结果。
    
    水系
        狭义的特指水系社区，广义的特指用于编写 :term:`类` 的程序系统。
